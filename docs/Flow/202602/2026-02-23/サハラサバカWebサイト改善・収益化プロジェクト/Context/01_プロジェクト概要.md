# プロジェクト概要

## サイトの目的

サハラマラソン（Marathon Des Sables）への挑戦を通じて、「小さな挑戦が大きな挑戦につながる。だから、まずは一歩踏み出してみる。」というメッセージを発信するWebサイト

## 運営者情報

- **名前**: 井上周（いのうえあまね）
- **職業**: スタートアップ企業のエンジニア
- **出身**: 群馬県（現在は東京）
- **専攻**: 大学で電気電子系を専攻

## コアメッセージ

**「一歩踏み出せない人」へ向けて**
- 大丈夫、一歩踏み出してみれば、サハラですら走りきれる
- だから、まずは小さな挑戦をしてみよう

## サハラマラソンとは

- **正式名称**: Marathon Des Sables（MDS）
- **開催地**: モロッコのサハラ砂漠
- **距離**: 250km
- **期間**: 7日間
- **特徴**:
  - 全ての衣食住を背負って走る
  - 「世の中で最も過酷なレース」
  - 日中の最高気温は50℃超
  - 起伏の多い厳しい環境

## 挑戦の背景

### きっかけ
1. 2023年夏：スペインのトマト祭りに参加
2. そこでサハラマラソン参加者と出会う
3. 「一度きりの人生で、サハラを走った人生とそうでない人生のどちらが良い？」という問いかけ
4. 2025年の参加を決意

### 哲学
- 人との出会いを大切にする
- 小さな挑戦の積み重ねが大きな挑戦につながる
- 「いつか」は来ない。少しでも興味があるならすぐやる
