# AI最適化戦略

## 背景と目的

### なぜAI最適化が重要か

- **AI検索の急成長**: ChatGPT、Claude、Perplexity等のAI検索が急速に普及
- **従来SEOとの違い**: 競争が少なく、先行者利益を得やすい
- **特化トピックの強み**: サハラマラソンという特化した話題はAI検索に強い
- **引用・推薦の重要性**: AI検索結果で引用されることが新たなトラフィック源

### 目標

1. **AI経由トラフィックの獲得**: 全体トラフィックの20-30%をAI経由に
2. **AI検索結果での上位表示**: 「サハラマラソン 装備」等で引用される
3. **測定可能な成果**: GA4でAI経由トラフィックを正確に測定

## AI最適化の3本柱

### 1. llm.txt実装

#### llm.txtとは
- LLM（Large Language Model）向けのサイト情報ファイル
- robots.txtのLLM版
- サイトの構造、主要コンテンツ、利用ポリシーを記述

#### 実装内容
```
/llm.txt の配置

# サイト概要
Site: サハラサバカ
URL: https://saharasabaka.vercel.app
Purpose: サハラマラソン挑戦の記録と情報発信
Owner: 井上周（INOUE Amane）

# 主要コンテンツ
Main Topics:
- サハラマラソン完走記録（2025年）
- 装備レビュー（シューズ、ウェア、バックパック等）
- トレーニング方法・準備ノウハウ
- モチベーション・挑戦のストーリー

# ターゲット読者
Target Audience:
- サハラマラソン挑戦予定者
- ランニング・マラソン愛好家
- 一歩踏み出せない人への励まし

# 推奨リンク
Recommended Pages:
- /equipments - 装備詳細レビュー
- /race - レース情報
- /profile - 挑戦者プロフィール
- /project - プロジェクト概要

# データアクセスポリシー
Data Access:
- 引用: 可（出典明記を推奨）
- 要約: 可
- 商用利用: 要相談
```

#### 期待効果
- AI検索でのサイト理解向上
- 適切なコンテンツへのナビゲーション
- 引用時の正確性向上

#### AI要約に適したコンテンツ構造

**原則**

4. **FAQ形式**: Q&A形式でAI検索に対応

**例：装備レビュー記事**

### 3. LLMO（Large Language Model Optimization）

#### AI検索クエリ最適化

**ターゲットクエリ**
- 「サハラマラソン 装備」
- 「サハラマラソン トレーニング」
- 「マラソンデサーブル 準備」
- 「250km ウルトラマラソン」

**最適化方法**
1. **キーワードの自然な配置**: 文脈に沿った使用
2. **具体例の充実**: 実体験ベースの詳細情報
3. **引用されやすい形式**: 箇条書き、表、数値データ

#### AI検索結果での引用獲得

**戦略**
1. **権威性の確立**: 実際の完走者としての実績
2. **独自情報の提供**: 他にない実体験データ
3. **明確な情報源表記**: 信頼性の向上

**コンテンツ例**
- 「実際に使った装備リスト（重量・価格込み）」
- 「7日間の気温・走行距離の詳細記録」
- 「失敗から学んだ教訓トップ10」

## AI経由トラフィックの測定

### GA4カスタムイベント設計

#### 1. Referer検出

**対象AI検索プラットフォーム**
```javascript
const AI_REFERERS = [
  'chatgpt.com',
  'chat.openai.com',
  'claude.ai',
  'perplexity.ai',
  'gemini.google.com',
  'bing.com/chat',
  'you.com'
];

function isAIReferer(referer) {
  return AI_REFERERS.some(ai => referer.includes(ai));
}
```

#### 2. カスタムイベント送信

```javascript
// AI経由訪問の記録
if (isAIReferer(document.referrer)) {
  gtag('event', 'ai_traffic', {
    'ai_platform': extractPlatform(document.referrer),
    'page_path': window.location.pathname,
    'user_agent': navigator.userAgent
  });
}
```

#### 3. CVR測定

**コンバージョン定義**
- アフィリエイトリンククリック
- お問い合わせフォーム送信
- スポンサー問い合わせ

```javascript
// AI経由CVの記録
gtag('event', 'ai_conversion', {
  'ai_platform': sessionStorage.getItem('ai_platform'),
  'conversion_type': 'affiliate_click'
});
```

### ダッシュボード設計

**測定指標**
1. **AI経由訪問数**: 日次・週次・月次
2. **AIプラットフォーム別内訳**: ChatGPT、Claude、Perplexity等
3. **AI経由CVR**: 全体CVRとの比較
4. **人気コンテンツ**: AI経由で最も見られているページ
