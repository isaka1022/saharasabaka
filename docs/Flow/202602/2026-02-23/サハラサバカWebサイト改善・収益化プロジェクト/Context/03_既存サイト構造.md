# 既存サイト構造

## ページ構成

### 主要ページ（8ページ）

| パス | コンポーネント | 目的 |
|------|---------------|------|
| `/` | HomePage | トップページ |
| `/profile` | ProfilePage | プロフィール紹介 |
| `/project` | ProjectPage | プロジェクト詳細 |
| `/race` | RacePage | レース情報 |
| `/equipments` | EquipmentsPage | 装備紹介 |
| `/support` | SupportPage | 応援・支援方法 |
| `/contact` | ContactPage | お問い合わせ |
| `/links` | Links | リンク集 |

## コンポーネント構成

### レイアウト
- **Header**: ナビゲーションヘッダー
- **Footer**: フッター

### ホームページ用コンポーネント
- **MainVisual**: メインビジュアル
- **Profile**: プロフィールセクション
- **Project**: プロジェクト紹介
- **Countdown**: カウントダウン
- **Members**: メンバー紹介
- **InstagramFeed**: Instagram連携
- **YouTubeEmbed**: YouTube埋め込み
- **Sponsors**: スポンサー表示

### ランディングページ
- **LandingPage/index**: ランディングページ統合
- **LandingPage/HeroSection**: ヒーローセクション
- **LandingPage/MotivationSection**: モチベーションセクション
